Roblox Plugin Quickstart

By ThisNoobPlays 2020.08.21
Learn how to create a simple Roblox plugin that you can expand on and add functionality to!

Prerequisites

  • Roblox scripting:
    • Variables
    • Strings
    • Arguments
    • Functions
    • Events
    • :Connect()

Instructions

  1. Make a new script in “ServerScriptStorage”
  2. Make a toolbar for your plugin by typing in on one line local toolbar = plugin:CreateToolbar("Plugin Name")
    Set the string in the parentheses to your plugin name.
  3. Make a button for your plugin by typing in on one line local button = toolbar:CreateButton("Button Name", "Button Description", "Button Icon")
    First argument : Button name that appears below the button
    Second argument: Description of button that appears on mouse hover
    Third argument: RbxAssetId link pointing to an image for your button icon
  4. Right click on your script object in the explorer and save as local plugin:

PluginScript > Save As Local Plugin

  1. Check that your plugin button is visible in the plugins toolbar
  2. Make a new function to house your plugin code. For example local function onButton() end
  3. Inside the function add your code / functionality
  4. Bind the button click to your function by typing this line after the function declaration : button.Click:Connect(onButton)
    Replace onButton with your function name
  5. Save your script as a local plugin again
  6. Test your plugin by clicking your plugin button
  7. Hopefully your plugin works!
